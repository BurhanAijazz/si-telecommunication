## 1. Ratio and Proportion

### What You Need to Know (Beginner Level)

A **ratio** compares two quantities of the same kind. A **proportion** states that two ratios are equal.

### Key Concepts

| Concept | Explanation | Example |
|---------|------------|---------|
| **Ratio** | Comparison of two quantities: a:b or a/b | If there are 3 boys and 5 girls, the ratio of boys to girls = **3:5** |
| **Proportion** | Two equal ratios: a:b = c:d | 2:3 = 4:6 (both simplify to 2:3) |
| **Simplest Form** | Divide both terms by their HCF | 12:18 = 2:3 (HCF of 12 and 18 is 6) |

### Important Formulas

| Formula | Usage |
|---------|-------|
| **a:b = a/b** | Basic ratio representation |
| **If a:b = c:d, then ad = bc** | Cross multiplication (product of extremes = product of means) |
| **If a:b and b:c are given, then a:b:c can be combined** | Make the common term (b) equal, then combine |
| **Dividing a quantity in ratio a:b:** Part 1 = Total x a/(a+b), Part 2 = Total x b/(a+b) | Splitting amounts |

### Worked Examples

**Example 1:** Divide Rs 1,500 between A and B in the ratio 2:3.
- Total parts = 2 + 3 = 5
- A's share = 1500 x (2/5) = **Rs 600**
- B's share = 1500 x (3/5) = **Rs 900**

**Example 2:** If a:b = 2:3 and b:c = 4:5, find a:b:c.
- Make "b" equal in both ratios. In the first ratio, b = 3. In the second, b = 4.
- LCM of 3 and 4 = 12.
- First ratio: a:b = 2:3 = 8:12 (multiply by 4)
- Second ratio: b:c = 4:5 = 12:15 (multiply by 3)
- Combined: **a:b:c = 8:12:15**

### Technical Deep-Dive: Types of Proportion

| Type | Meaning | Example |
|------|---------|---------|
| **Direct Proportion** | When one quantity increases, the other also increases at the same rate | More workers = more work done (in the same time) |
| **Inverse Proportion** | When one quantity increases, the other decreases | More workers = less time to finish the same work |

## 2. Percentages

### What You Need to Know (Beginner Level)

**Percentage** means "per hundred." It is a way of expressing a number as a fraction of 100.

### Key Formulas

| Formula | Usage |
|---------|-------|
| **Percentage = (Value / Total) x 100** | Finding what percentage one number is of another |
| **Value = (Percentage / 100) x Total** | Finding the value from a percentage |
| **Percentage Increase = [(New - Old) / Old] x 100** | Calculating how much something has increased |
| **Percentage Decrease = [(Old - New) / Old] x 100** | Calculating how much something has decreased |

### Fraction-Percentage Equivalents (Must Memorize)

| Fraction | Percentage | Fraction | Percentage |
|:--------:|:----------:|:--------:|:----------:|
| 1/2 | 50% | 1/8 | 12.5% |
| 1/3 | 33.33% | 1/9 | 11.11% |
| 1/4 | 25% | 1/10 | 10% |
| 1/5 | 20% | 1/12 | 8.33% |
| 1/6 | 16.67% | 2/3 | 66.67% |
| 1/7 | 14.28% | 3/4 | 75% |

### Worked Examples

**Example 1:** A shirt costs Rs 800. After a 15% discount, what is the selling price?
- Discount = 15% of 800 = (15/100) x 800 = Rs 120
- Selling Price = 800 - 120 = **Rs 680**

**Example 2:** The population of a town increased from 50,000 to 55,000. What is the percentage increase?
- Increase = 55,000 - 50,000 = 5,000
- Percentage Increase = (5,000 / 50,000) x 100 = **10%**

**Example 3 (Successive Percentages):** A price increases by 20% and then decreases by 10%. What is the net percentage change?
- Let the original price = 100.
- After 20% increase: 100 + 20 = 120.
- After 10% decrease on 120: 120 - 12 = 108.
- Net change = 108 - 100 = **8% increase**

> **Shortcut Formula for Successive Percentage Change:**
> Net % change = a + b + (a x b)/100
> = 20 + (-10) + (20 x -10)/100 = 20 - 10 - 2 = **8%**

## 3. Profit and Loss

### What You Need to Know (Beginner Level)

When you buy something (Cost Price) and sell it (Selling Price), you either make a **Profit** or incur a **Loss**.

### Key Terms and Formulas

| Term | Meaning |
|------|---------|
| **Cost Price (CP)** | The price at which an item is bought |
| **Selling Price (SP)** | The price at which an item is sold |
| **Profit** | SP - CP (when SP > CP) |
| **Loss** | CP - SP (when CP > SP) |
| **Marked Price (MP)** | The listed/labeled price before any discount |
| **Discount** | MP - SP |

| Formula | Expression |
|---------|-----------|
| **Profit %** | (Profit / CP) x 100 |
| **Loss %** | (Loss / CP) x 100 |
| **SP when Profit % is given** | SP = CP x (100 + Profit%) / 100 |
| **SP when Loss % is given** | SP = CP x (100 - Loss%) / 100 |
| **Discount %** | (Discount / MP) x 100 |
| **SP after Discount** | SP = MP x (100 - Discount%) / 100 |

### Worked Examples

**Example 1:** A shopkeeper buys a watch for Rs 500 and sells it for Rs 600. Find the profit percentage.
- Profit = 600 - 500 = Rs 100
- Profit % = (100 / 500) x 100 = **20%**

**Example 2:** An article is marked at Rs 1,200. A discount of 15% is offered. Find the selling price.
- SP = 1200 x (100 - 15) / 100 = 1200 x 85 / 100 = **Rs 1,020**

**Example 3:** A man sells two articles, each for Rs 990. On one, he gains 10%, and on the other, he loses 10%. Find the overall profit or loss.
- Article 1: SP = 990, Profit = 10%. So CP = 990 x 100/110 = Rs 900
- Article 2: SP = 990, Loss = 10%. So CP = 990 x 100/90 = Rs 1,100
- Total CP = 900 + 1100 = Rs 2,000
- Total SP = 990 + 990 = Rs 1,980
- **Loss = Rs 20, Loss % = (20/2000) x 100 = 1%**

> **Key Rule:** When the same selling price gives equal profit% and loss% on two items, there is **always a net loss**. Quick formula: Loss % = (Common %)^2 / 100 = (10)^2 / 100 = **1%**

## 4. Simple and Compound Interest

### What You Need to Know (Beginner Level)

**Interest** is the cost of borrowing money (or the reward for saving/investing money).

### Key Terms

| Term | Meaning |
|------|---------|
| **Principal (P)** | The original amount of money borrowed or invested |
| **Rate (R)** | The percentage of interest charged per year |
| **Time (T)** | The duration for which the money is borrowed/invested (in years) |
| **Simple Interest (SI)** | Interest calculated only on the original principal |
| **Compound Interest (CI)** | Interest calculated on the principal AND the accumulated interest |

### Formulas

| Type | Formula |
|------|---------|
| **Simple Interest** | **SI = (P x R x T) / 100** |
| **Amount (SI)** | **A = P + SI = P(1 + RT/100)** |
| **Compound Interest** | **A = P(1 + R/100)^T** |
| **CI** | **CI = A - P = P[(1 + R/100)^T - 1]** |
| **CI compounded half-yearly** | A = P(1 + R/200)^(2T) (rate halved, time doubled) |
| **CI compounded quarterly** | A = P(1 + R/400)^(4T) (rate quartered, time x4) |

### Comparison: SI vs. CI

| Feature | Simple Interest | Compound Interest |
|---------|----------------|-------------------|
| **Interest calculated on** | Original principal only | Principal + accumulated interest |
| **Interest each year** | Same every year | Increases each year |
| **Formula** | SI = PRT/100 | A = P(1 + R/100)^T |
| **When is CI > SI?** | Always (except for T = 1 year, when they are equal with annual compounding) | -- |

### Worked Examples

**Example 1 (Simple Interest):** Find the SI on Rs 5,000 at 8% per annum for 3 years.
- SI = (5000 x 8 x 3) / 100 = **Rs 1,200**
- Amount = 5000 + 1200 = **Rs 6,200**

**Example 2 (Compound Interest):** Find the CI on Rs 10,000 at 10% per annum for 2 years, compounded annually.
- A = 10000 x (1 + 10/100)^2 = 10000 x (1.1)^2 = 10000 x 1.21 = Rs 12,100
- CI = 12100 - 10000 = **Rs 2,100**

**Example 3 (Difference between CI and SI for 2 years):**
> **Shortcut:** Difference = P x (R/100)^2
- For P = 10000, R = 10: Difference = 10000 x (10/100)^2 = 10000 x 0.01 = **Rs 100**
- Verification: SI for 2 years = (10000 x 10 x 2)/100 = Rs 2,000. CI = Rs 2,100. Difference = Rs 100. Confirmed.

## 5. Averages

### What You Need to Know (Beginner Level)

An **average (mean)** is the sum of all values divided by the number of values. It represents the central value of a data set.

### Key Formulas

| Formula | Expression |
|---------|-----------|
| **Average** | **Average = Sum of all values / Number of values** |
| **Sum** | **Sum = Average x Number of values** |
| **Weighted Average** | **(w1.x1 + w2.x2 + ...) / (w1 + w2 + ...)** |

### Important Properties

| Property | Explanation |
|----------|------------|
| **If all values are equal** | The average equals that value |
| **Adding a new value greater than the current average** | The new average increases |
| **Adding a new value less than the current average** | The new average decreases |
| **Average of first n natural numbers** | (n + 1) / 2 |
| **Average of first n even numbers** | (n + 1) |
| **Average of first n odd numbers** | n |
| **Average of consecutive numbers from a to b** | (a + b) / 2 |

### Worked Examples

**Example 1:** The marks of 5 students are 72, 85, 90, 68, and 95. Find the average.
- Sum = 72 + 85 + 90 + 68 + 95 = 410
- Average = 410 / 5 = **82**

**Example 2:** The average age of 30 students is 15 years. If the teacher's age is included, the average increases by 1. What is the teacher's age?
- Sum of ages of 30 students = 30 x 15 = 450
- New average (31 people) = 16
- New sum = 31 x 16 = 496
- Teacher's age = 496 - 450 = **46 years**

**Example 3:** The average of 5 numbers is 20. If one number is excluded, the average becomes 18. What is the excluded number?
- Sum of 5 numbers = 5 x 20 = 100
- Sum of remaining 4 numbers = 4 x 18 = 72
- Excluded number = 100 - 72 = **28**

## 6. Time, Speed and Distance

### What You Need to Know (Beginner Level)

These problems involve the relationship between **distance**, **speed**, and **time**.

### Key Formulas

| Formula | Expression |
|---------|-----------|
| **Speed** | **Speed = Distance / Time** |
| **Distance** | **Distance = Speed x Time** |
| **Time** | **Time = Distance / Speed** |

### Unit Conversions

| Conversion | Formula |
|-----------|---------|
| **km/h to m/s** | Multiply by **5/18** |
| **m/s to km/h** | Multiply by **18/5** |

**Example:** 72 km/h = 72 x 5/18 = **20 m/s**

### Important Concepts

**1. Average Speed:**

| Scenario | Formula |
|----------|---------|
| Same distance at speeds S1 and S2 | **Average Speed = 2 x S1 x S2 / (S1 + S2)** |
| Different distances d1 and d2 at speeds S1 and S2 | **Average Speed = (d1 + d2) / (d1/S1 + d2/S2)** |

> **Important:** Average speed is NOT the simple average of two speeds (unless the times are equal, not the distances).

**2. Relative Speed:**

| Scenario | Relative Speed |
|----------|---------------|
| **Same direction** | S1 - S2 (faster - slower) |
| **Opposite direction** | S1 + S2 |

**3. Trains:**

| Scenario | Distance Covered |
|----------|-----------------|
| Train crossing a pole/person | Length of the train |
| Train crossing a platform/bridge | Length of train + Length of platform |
| Two trains crossing each other (opposite direction) | Sum of both lengths; Relative speed = S1 + S2 |
| Two trains crossing each other (same direction) | Sum of both lengths; Relative speed = S1 - S2 |

### Worked Examples

**Example 1:** A car travels 240 km in 4 hours. What is its speed?
- Speed = 240 / 4 = **60 km/h**

**Example 2:** A man covers half the distance at 40 km/h and the other half at 60 km/h. What is his average speed?
- Average Speed = 2 x 40 x 60 / (40 + 60) = 4800 / 100 = **48 km/h**

**Example 3:** A train 150 m long is running at 72 km/h. How long will it take to cross a platform 250 m long?
- Total distance = 150 + 250 = 400 m
- Speed = 72 km/h = 72 x 5/18 = 20 m/s
- Time = 400 / 20 = **20 seconds**

## 7. Elementary Statistics and Data Interpretation

### What You Need to Know (Beginner Level)

**Statistics** is the branch of mathematics dealing with the collection, analysis, and interpretation of data. **Data Interpretation (DI)** tests your ability to read and analyze data presented in tables, charts, or graphs.

### 7.1 Measures of Central Tendency

| Measure | Definition | Formula |
|---------|-----------|---------|
| **Mean (Average)** | Sum of all observations divided by the number of observations | Mean = Sum / n |
| **Median** | The middle value when data is arranged in order | If n is odd: middle value. If n is even: average of two middle values |
| **Mode** | The value that appears most frequently | The observation with the highest frequency |

**Example:** Data set: 3, 7, 7, 2, 9, 1, 7
- **Mean** = (3+7+7+2+9+1+7) / 7 = 36/7 = 5.14 (approx.)
- **Median** = Arrange: 1, 2, 3, **7**, 7, 7, 9. Middle value = **7**
- **Mode** = **7** (appears 3 times, more than any other value)

### 7.2 Range

**Range** = Highest value - Lowest value

From the example above: Range = 9 - 1 = **8**

### 7.3 Data Interpretation (Types of Charts)

| Chart Type | When Used | How to Read |
|-----------|-----------|-------------|
| **Bar Graph** | Comparing quantities across categories | Height of each bar represents the value |
| **Pie Chart** | Showing parts of a whole (percentages) | Each slice represents a proportion; total = 360 degrees = 100% |
| **Line Graph** | Showing trends over time | Points connected by lines; rising line = increase, falling = decrease |
| **Table** | Presenting exact numerical data | Read row and column headers carefully |

### Pie Chart Calculations

| Conversion | Formula |
|-----------|---------|
| **Percentage to degrees** | Degrees = (Percentage / 100) x 360 |
| **Degrees to percentage** | Percentage = (Degrees / 360) x 100 |
| **Value from percentage** | Value = (Percentage / 100) x Total |

**Example:** In a pie chart of a company's expenses totaling Rs 2,00,000:
- Salaries = 40% = (40/100) x 200000 = **Rs 80,000**
- Salaries sector angle = (40/100) x 360 = **144 degrees**

### 7.4 Strategy for Data Interpretation

1. **Read the title** and labels carefully.
2. **Understand units** (thousands, lakhs, crores, percentages).
3. **Use approximation** where possible -- do not waste time on exact calculations.
4. **Check all options** before finalizing.
5. **Be careful with percentage questions** -- identify the base (of what?).

### Worked DI Example

**Table: Sales (in lakhs) of Company X over 5 years**

| Year | Sales (Rs Lakhs) |
|:----:|:-----------------:|
| 2020 | 120 |
| 2021 | 150 |
| 2022 | 180 |
| 2023 | 210 |
| 2024 | 250 |

**Q:** What is the percentage increase in sales from 2020 to 2024?
- Increase = 250 - 120 = 130
- % Increase = (130 / 120) x 100 = **108.33%**

**Q:** What is the average sales over the 5 years?
- Total = 120 + 150 + 180 + 210 + 250 = 910
- Average = 910 / 5 = **Rs 182 lakhs**
